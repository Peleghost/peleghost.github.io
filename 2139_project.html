C# MVC Application simulating an online store, with features such as:

- Implementation of Microsoft Identity.

- Login and Registration with email confirmation, sending the email through an SMTP client.

- Forgot Password with token sent via email for password reset.

- Ability to search for products with working filters.

- Order placement and details page.

- Admins may create new product to add to the store.

- Authentication and Authorization implemented on the controllers to dictate which elements users have access to.
