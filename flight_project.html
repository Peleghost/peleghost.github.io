COMP2129 Advanced OOP - Flight Management App

Console based application developed using C#. With features such as:

- Customer CRUD with validation for name and phone number

- Flight CRUD with validation and unique id generation

- Flight Booking CRUD

- All information is saved to a file with the implementation of a FileManager class that is resposible for all file operations and is injected into the required classes
